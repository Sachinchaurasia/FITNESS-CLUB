Online Examination System is a MCQ(Multiple Choice Question) based examination system which is built using JAVA and MySQL. It provides an easy to use Interface for both Test Conductor and student appering for the examination.

Mission

  • REsponse by the candidate will be checked automatically and instantly.
  • Online Examination will reduce the hectic job of assessing the answers given by the candidates.
  • This project would be very useful for educational institutes where regular evaluation of student is required.

Values

  • Customer Satisfaction
  • Team Work
  • Integrity
  • Timely response which provide solutions and service.
